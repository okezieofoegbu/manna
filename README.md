# Manna

*Word before work.*

A private morning page: a slow devotional engine, and (from v0.1.3) a curated
brief of the Transworld inbox.

- **Current version:** v0.1.2 — the devotional engine (Step 2 of Phase 1).
- **Stack:** Next.js · Supabase · Vercel · GitHub · Anthropic API · API.Bible
- **Run locally:** `npm run dev` (local only — there is no auth gate yet).

See the four handover documents — `INSTRUCTIONS.md`, `STATE_OF_APP.md`,
`DB_SCHEMA.md`, `PITFALLS.md` — for the full picture.
